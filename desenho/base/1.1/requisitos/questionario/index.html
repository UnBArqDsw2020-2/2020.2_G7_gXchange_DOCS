    Questionário

    Introdução

    O método de Questionário consiste na criação de um conjunto de perguntas, o principal motivo de utilizarmos a técnica de questionário foi devido ao seu baixo custo atrelado à uma quantidade de respostas razoável, além de tornar possível a obtenção de respostas de diversos possíveis usuários em posições geográficas diferentes e a partir disso, elicitar os requisitos de acordo com suas respostas.

    Requisitos Elicitados

    Funcionais

    Número Requisito
    1 O usuário deve ser capaz de visualizar as fotos de um anúncio
    2 O usuário deve ser capaz de enviar mensagens para outros usuários
    3 O usuário deve ser capaz de acessar uma página de ajuda e suporte
    4 O usuário deve ser capaz de avaliar outros usuários
    5 O usuário deve ser capaz de visualizar o perfil de outros usuários
    6 O usuário deve ser capaz de visualizar o feed de anúncios
    7 O usuário deve ser capaz de buscar por um anúncio
    8 O usuário deve ser capaz de realizar o seu cadastro por email
    9 O usuário deve ser capaz de realizar o seu cadastro pelo facebook
    10 O usuário deve ser capaz de realizar o seu cadastro pelo google
    11 O usuário deve ser capaz de visualizar as avaliações de outros usuários
    12 O usuário deve ser capaz de visualizar os dados (nome do jogo, tempo de uso, plataforma, localização, preço) de um anúncio

    Não Funcionais

    Número Requisito
    1 O sistema deve manter a senha do usuário encriptada
    2 O sistema deve ser capaz de responder às requisições do usuário rapidamente (a ser decidido)
    3 O sistema deve ser confiável (a ser decidido)
    4 O sistema deve ser de fácil aprendizado (a ser decidido)
    5 O sistema deve dispor os anúncios de maneira organizada (a ser decidido)
